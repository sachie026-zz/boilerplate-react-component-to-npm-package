# react-multi-step-form-component

A NPM package which lets user add their personal components as a step


Installation:

npm install multi-step-form-ui 
or 
yarn add multi-step-form-ui


Usage:

import MultiSteps from "multi-step-form-ui";

 <MultiSteps>
    <Component1 />
    <Component2 />
    <Component3/>
</MultiSteps>
